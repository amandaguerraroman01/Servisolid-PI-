# Servisolid-PI-
O projeto ServiSolid tem como objetivo conectar prestadores de serviços, contratantes e instituições filantrópicas em uma plataforma digital segura e ética. Dessa forma, o nosso grupo desenvolveu um sistema digital capaz de unir dois mundos: o da geração de renda e o do voluntariado.
